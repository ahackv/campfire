# Campus Voice (Hackathon MVP)

Campus Voice is a youth civic product with two connected experiences:

1. **Government debate hub**: students vote/comment on real policy debates.
2. **Your Town, Your Choices game**: a short civic simulation that teaches consequences.

## What changed from the previous version

- Removed the passive comparison section and focused on a clean, direct UX.
- Reduced button overload by using fewer primary actions and select-based choices.
- Updated debates/polls to represent real government discussion topics.
- Added visual dashboard graphs for support and participation trends.
- Added a unique hackathon concept: **Youth Mandate Brief** (one-click decision summary for adults and institutions).

## Why this can win

- It is easy for students and adults to understand quickly.
- It combines engagement + evidence + decision-ready outputs.
- It avoids being just another “issue posting wall” by linking participation to policy-ready briefs.

## MVP features

- Student profile (local demo)
- Debate feed tied to government topics
- Debate room: vote support/oppose/amend + comments
- One-click Youth Mandate Brief
- Insight graphs (issue support + participation trend)
- 4-round civic simulation game with outcomes
